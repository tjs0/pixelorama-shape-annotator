# Installing the Shape Annotator extension

1. **Get the `.pck`** — download `ShapeAnnotator.pck` from the releases page, or build it
   yourself: from the repo root run `./build.sh` (requires Godot 4.x + export templates),
   which writes `dist/ShapeAnnotator.pck`.
2. **Copy it into Pixelorama's extensions folder.** In Pixelorama: **Help ▸ Open User Data
   Folder**, then put `ShapeAnnotator.pck` in the `extensions/` subfolder (create it if
   missing). The file name must stay `ShapeAnnotator.pck`.
3. **Enable it** under **Edit ▸ Preferences ▸ Extensions** → toggle **ShapeAnnotator** on.
4. **Restart** Pixelorama if prompted.
